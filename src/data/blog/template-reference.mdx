# MDX Blog Template Reference

Use this file as a copy-paste reference whenever you forget syntax.

<Callout type="warning" title="Internal template">
  This file is prefixed with <code>_</code>, so it will not be published by your current content glob.
  Remove the prefix and set <code>draft: false</code> only when you intentionally want to publish it.
</Callout>

## 1) Frontmatter template

```yaml
---
title: "Your Post Title"
author: Vetcha Pankaj Nath
pubDatetime: 2026-01-01T10:00:00Z
description: "Short summary for cards, SEO, and social previews."
draft: false
tags: ["tag-1", "tag-2"]
featured: false
modDatetime: 2026-01-02T18:30:00Z
canonicalURL: "https://example.com/original-post"
hideEditPost: false
timezone: "Asia/Kolkata"
---
```

## 2) Text formatting

Normal text, **bold**, *italic*, ***bold+italic***, ~~strikethrough~~, and `inline code`.

You can also use HTML where needed: <mark>highlighted text</mark>, H<sub>2</sub>O, 10<sup>6</sup>.

Current year via MDX expression: **{new Date().getFullYear()}**.


Code block with filename label (uses your custom `file="..."` meta):

```ts file="src/utils/example.ts"
export function sum(a: number, b: number) {
  return a + b;
}
```

Shiki notation examples (highlight, word highlight, diff):

```ts file="src/config/feature.ts"
const apiBase = "https://old.api.dev"; // [!code --]
```
